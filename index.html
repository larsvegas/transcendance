What is Transcendance?

Transcendance is a plugin for jQuery. With it, you can cycle images on your website.

Why choose Transcendance?

There are a lot of image cycle plugins for jQuery on the web. Transcendance, however, offers it's users a variety of customizing the way images are being cycled. In the above example you find the options you can define within Transcendance. Further more, Transcendance is compatible with most browsers, such as Firefox, Chrome, Safari, Opera and Internet Explorer.

What can it do?

For instance, you can define in how many blocks, called cells, your images are being split. You can define how long it takes a cell to fade in and how long it will take to go over to the next cell to fade in. Additionally, you can set how long an image will be visible before transcending (caution: word play) to the next image. If you want more randomness, you can set the cell sort order to random which will randomly pick cells to fade in. You can also set the image order itself to be random.
